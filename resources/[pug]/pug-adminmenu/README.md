## ⚙️ Installation Instructions (the documentation page above explains this better)
1. Drag, drop, ensure.
2. Set yourself as the god role. Add this convar anywhere in your server.cfg and add your fivem identifier to it. `set admin_menu_god "identifier.fivem:12345"`
4. Remove `qb-adminmenu` if you have this in your server.
5. After you load into the server you want to manually set your staffs admin roles from within the admin menu itself. Find there profiles and click `set perms`
6. (OPTIONAL) To allow Fivemanage logs set this inside of your server.cfg `set FIVEMANAGE_LOGS_API_KEY "YOUR_REAL_FIVEMANAGE_KEY"`


## 🔐 Keybinds (Configurable in the config.lua)
1. F2 to noclip
2. F3 to open the menu
